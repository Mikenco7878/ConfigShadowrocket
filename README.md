# Shadowrocket Pro

🚀 Bộ sưu tập module, script và config dành cho Shadowrocket trên iOS.

## ✨ Tính năng

- Chặn quảng cáo
- Rewrite nâng cao
- Script unlock
- Tối ưu tốc độ mạng
- Hỗ trợ module riêng cho nhiều ứng dụng
- Tự động cập nhật script từ GitLab Raw


---

## ⚙️ Hướng dẫn sử dụng

1. Mở Shadowrocket
2. Import module
3. Bật MITM
4. Cài chứng chỉ HTTPS
5. Khởi động lại ứng dụng cần dùng

---

## 🛠 Hỗ trợ

Nếu module lỗi:
- Kiểm tra MITM
- Kiểm tra hostname
- Đảm bảo script raw hoạt động
- Thử đổi mạng/VPN

---

## 📱 Yêu cầu

- iOS 10+
- Shadowrocket mới nhất
